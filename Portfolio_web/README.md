# Portfolio-sivusto – Lea Salonen

Tämä työ on tehty keväällä 2026 osana **Websivuston kehittäminen** -kurssia.

Sivuston tavoitteena on toimia Lea Salosen portfoliona, johon on koottu tehtyjä projekteja sekä opittuja asioita.

## Sivuston rakenne

### Etusivu
- Kuva
- Otsikko (nimi)
- Lyhyt esittely

### Projektit
Jokainen projekti sisältää:
- Projektin nimi
- Ongelma ja sen tunnistaminen
- Tutkimus ja päätöksenteko
- Projektin scope: miksi tähän ratkaisuun ja järjestykseen päädyttiin
- Tulokset, kuten käyttöaste, kaupallisuus ja jatkokehitys
- Mitä opin ja mitä tekisin toisin
