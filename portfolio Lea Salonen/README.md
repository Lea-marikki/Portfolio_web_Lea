# Portfolio — Lea Salonen

Portfoliosivu kurssityönä, Laurea-ammattikorkeakoulu.

## Lähteet ja viitteet

**Kurssimateriaali**
- Laurea-ammattikorkeakoulu. Websivuston kehittämisen kurssin materiaalit, luennot ja harjoitustehtävät. 2025–2026. Berfin ja Jari

**Inspiraatio ja layout**
- Sitebuilder Report. Product Manager Portfolio Examples. Haettu 27.4.2026. https://www.sitebuilderreport.com/inspiration/product-manager-portfolios

**UX ja käytettävyys**
- Wang, H. (2024, 15. maaliskuuta). Homepage Design: 5 Fundamental Principles. Nielsen Norman Group. Haettu 27.4.2026. https://www.nngroup.com/articles/homepage-design-principles/
  - Viisi periaatetta: helppo pääsy etusivulle, selkeä viestintä, sisältöesimerkit, toimintakehotteet ja yksinkertaisuus. Käytetään arviointikriteerinä sivuston toimivuuden arvioinnissa.

**Tekniset kirjastot**
- Bootstrap 5.3. (2024). Bootstrap. https://getbootstrap.com
- Google Fonts. (2024). DM Serif Display & Inter. https://fonts.google.com

**Tekoälyavusteisuus**
- Anthropic Claude (malli: claude-sonnet-4-6) on toiminut kehitysapuna sivuston rakentamisessa Claude Code -työkalun kautta. Tekoälyä on käytetty seuraaviin tarkoituksiin:
  - HTML- ja CSS-koodin kirjoittaminen ja rakentaminen promptauksen perusteella
  - Sivurakenteen ja visuaalisen ilmeen ideointi
  - Sisällön muotoilu ja tekstien jäsentely CV:n ja Confluence-dokumentaation pohjalta
  - Tiedostorakenteen ja kansioinnin suunnittelu
- Kaikki sisältö (projektikuvaukset, urapolku, osaaminen) perustuu tekijän omiin tietoihin ja dokumentteihin. Koodi on katselmoitu ja hyväksytty tekijän toimesta ennen committointia.
